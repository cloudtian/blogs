让HTML输出文本不转义

如果有空格和回车这样的特殊字符的简单文本，可以在文本外加pre标签

            start
            用pre标签包裹着的内容
                 前面有五个空格也能全部显示
            无法显示空白字符：( )
            end

如果是HTML，JavaScript，xml或其他特殊内容，可以使用xmp标签（不过该元素已被废弃）

start 用xmp标签包裹着的内容 前面有五个空格也能全部显示 可以显示空白字符：(&nbsp;) 空白字符有&#32;&nbsp;&emsp;&ensp;等等 end

通过&amp;来转义特殊字符中的&，从而达到显示特殊字符的效果。例如：&#32;

特殊字符使用标签展示例子

xmp标签显示:&#32;|&nbsp;|&ensp;|&#8194;|&emsp;|&#8195;|&#12288;|&#8203; 
pre标签显示: | | | | | |　|​
code标签显示: | | | | | |　|​

generic-font字体显示: | | | | | |　|​

xmp标签之间的内容不会被当作文档内容解析，而会被用等宽字体直接呈现。

mdn上：该元素已被废弃。建议使用pre和code元素。通过css样式设置font-family属性为generic-font，可以让任何其他标签获得等宽字体样式。 然而，从上面的例子可以看出，只有xmp标签才能显示键盘的空白字符

&nbsp;|&#160;

unicode U+00A0 普通的英文半角空格但不换行，no-break space

这个空格本来的使用场景是：用在需要避免断行的地方，比如: 123 MB数字与单位之间不应该有换行

这个字符其实就相当于一个英文字母或数字式，受到跟字体相关的样式的影响，比如：font-family,font-size,font-weight等。不同浏览器不同字体等出来的效果都是不一致的。
