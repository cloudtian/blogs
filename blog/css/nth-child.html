nth-child

x:nth-child(n): 属于其父元素的第n个子元素的每个x

计算方式：1.计算同父节点的第几个节点，2.判断选择器

            /* nth-child的第2个子元素且为.b时显示红色，但当前第2个子元素是a所以没效果*/
            .nth-child .b:nth-child(2) {
                background-color: blue;
            }
            /*第四个子元素为b 所以有效果*/
            .nth-child .b:nth-child(4) {
                background-color: green;
            }
            /*奇数的子元素且为b则显示*/
            .nth-child .b:nth-child(odd) {
                background-color: bisque;
            }

nth-of-type

x:nth-of-type(n): 属于其父元素的特定元素的第n个节点的每个x

计算方式：1.先将同父节点按照标签分组，2.按照分组后的标签是第几个节点，3.判断选择器

            /*nth-of-type的子元素中按标签分组后，每组的第2个元素是b的，但当前是a，所以没效果*/
            .nth-of-type .b:nth-of-type(2) {
                background-color: darkmagenta;  
            }
            .nth-of-type .b:nth-of-type(3) {
                background-color: darkcyan;  
            }
            .nth-of-type .b:nth-of-type(6) {
                background-color: yellow;
            }
